Föreläsare och Konferens

Här hittar du en lista över tillgängliga föreläsare och konferenser hos Boka Konferens. Du kan läsa mer om varje specifik föreläsare och konferens längre ner på sidan.

Anmäla

Colin Moon

Responsive Web Design Masterclass

I denna masterclass fördjupar du din förståelse för responsiv webbdesign. Utforska avancerade tekniker för att skapa webbplatser som anpassar sig smidigt till olika skärmstorlekar och enheter.

Tommy Lundberg

Frontend Security Essentials

Lär dig hur du säkrar din frontend-kod och skyddar webbapplikationer mot sårbarheter och hot. Detta seminarium fokuserar på bästa säkerhetspraxis för frontend-utvecklare.

Gunnar Ekman

Web Performance Summit

Web Performance Summit samlar experter inom webbprestandaoptimering. Ta del av djupgående sessioner om hastighetsoptimering, cachning och lasttestning för webbplatser och appar.

Eric Thyrell

Frontend Fusion 2023

Frontend Fusion är årets höjdpunkt för frontend-utvecklare. Anslut till branschledande experter för att lära dig om de senaste trenderna inom webbutveckling, JavaScript-ramverk och användarupplevelseoptimering.

Eric Donell

ReactRush 2023

ReactRush är en tvådagarskonferens som fokuserar helt och hållet på React och dess ekosystem. Få insikter från utvecklare bakom populära React-projekt och ta del av praktiska workshoppar.

Ami Hemviken

CSS Creativity Conference

CSS Creativity Conference utforskar konst och kreativitet i webbutveckling. Upptäck avancerade CSS-tekniker, animationer och designprinciper som hjälper dig skapa imponerande webbgränssnitt.

Annmarie Palm

Progressive Web Apps Workshop

Detta praktiska seminarium fokuserar på att bygga Progressive Web Apps (PWAs) från grunden. Lär dig hur du skapar webbappar som fungerar snabbt och smidigt på olika enheter och plattformar.

Ebba Lourin

JavaScript Frameworks Crash Course

Det här intensiva seminariet är en snabbkurs i populära JavaScript-ramverk som Angular, Vue och React. Få en översikt över deras funktioner och användningsområden.

Emma Pihl

UX/UI Design Expo

UX/UI Design Expo är en tvådagarskonferens som utforskar gränssnittsdesign och användarupplevelse. Upptäck de senaste verktygen och teknikerna för att skapa engagerande användargränssnitt.

Robert Karjel

Web Accessibility Summit

Web Accessibility Summit är en konferens som fokuserar på att göra webben tillgänglig för alla. Lär dig bästa praxis och tekniker för att skapa webbplatser och appar som är tillgängliga och användbara för personer med funktionsnedsättningar.

Om oss

Boka konferens förmedlar föreläsningar för alla tillfällen. Oavsett om du behöver en föreläsare, underhållare, moderator eller konferencier har vi någon som passar för ditt specifika evenemang. Vi hanterar arrangemang av alla storlekar, både offentliga och privata, för studenter och skolor. Vi fungerar som länken mellan dig och din talare.
